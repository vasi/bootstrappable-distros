# Gloire

## Bootstrapping info

Upstream: https://www.codeberg.org/Ironclad/Gloire

Requirements: Linux-based system and some standard UNIX tools.

Automation: Yes, using the Jinx build system

Integration testing: No

What you get: A full Gloire desktop! With any mix of packages you may want. By
default, a terminal-only minimal install.

Can install further software from source: Yes, by either using `xbps` from
inside the distro or `PKGS_TO_INSTALL` when building the image.

## Manual testing
